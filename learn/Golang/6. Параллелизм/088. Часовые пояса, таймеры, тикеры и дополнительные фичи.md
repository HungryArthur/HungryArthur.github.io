#### Часовые пояса — тип time.Location
В Go часовые пояса представлены типом `*time.Location`. Это не просто смещение от UTC — это полноценная структура, которая знает всё о конкретной временной зоне: текущее смещение, исторические изменения, переходы на летнее/зимнее время и даже их отмену (как это было в некоторых странах). Go использует официальную базу IANA (tzdata) — ту же, что и Linux, macOS и большинство серверов. Благодаря этому вы можете быть уверены: если где-то в мире перевели часы — Go это знает.

#### Готовые пояса: time.UTC и time.Local
`time.UTC` — это «чистое» универсальное время без какого-либо смещения. Используется в логировании, API, базах данных. `time.Local` — это ваш локальный часовой пояс, который берётся из настроек операционной системы. Это удобно, когда нужно показать пользователю время «как у него на компьютере».
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    now := time.Now()
    utc := now.In(time.UTC)
    local := now.In(time.Local)
    
    fmt.Println("В UTC:", utc.Format("15:04:05"))
    fmt.Println("В локальном поясе:", local.Format("15:04:05"))
}
```
**Вывод (пример, если вы в часовом поясе Москвы):**
```yaml
В UTC: 11:22:10
В локальном поясе: 14:22:10
```

#### Загрузка пояса по имени: time.LoadLocation
Если нужно точное время в конкретном городе — используйте IANA-название (например, `Europe/Moscow`, `America/New_York`, `Asia/Tokyo`). Полный список можно посмотреть [здесь](https://en.wikipedia.org/wiki/List_of_tz_database_time_zones).
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    moscow, err := time.LoadLocation("Europe/Moscow")
    if err != nil {
        fmt.Println("Ошибка загрузки пояса:", err)
        return
    }
    
    now := time.Now().In(moscow)
    fmt.Println("Сейчас в Москве:", now.Format("15:04:05 02.01.2006"))
}
```
**Вывод (пример):**
```undefined
Сейчас в Москве: 14:22:10 06.12.2025
```

#### Создание фиксированного пояса: time.FixedZone
Если вам нужен простой пояс без переходов на летнее время (например, для тестов или специфических регионов), используйте `time.FixedZone`. Второй аргумент — смещение в секундах от UTC.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    // Нью-Йорк зимой — UTC-5
    ny := time.FixedZone("America/New_York", -5*60*60)
    now := time.Now().In(ny)
    fmt.Println("Сейчас в Нью-Йорке (фиксированный пояс):", now.Format("15:04:05"))
}
```
**Вывод (пример):**
```scss
Сейчас в Нью-Йорке (фиксированный пояс): 06:22:10
```

#### Таймеры и тикеры — когда нужно «подождать» или «делать что-то регулярно»
В реальных приложениях очень часто нужно:
- Ограничить время ожидания ответа от сервера
- Запускать фоновые задачи каждые N секунд/минут
- Делать «heartbeat» (пинг) каждые 30 секунд
- Обновлять что–либо каждые 5 минут

Для этого в пакете `time` есть мощный набор инструментов: от простого `Sleep` до гибких таймеров и тикеров.

#### time.Sleep — простая пауза
Самый простой способ «подождать», который мы много раз использовали до этого в курсе. Блокирует текущую горутину на указанное время.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    fmt.Println("Начинаем ждать...")
    time.Sleep(2 * time.Second)
    fmt.Println("Прошло 2 секунды — продолжаем!")
}
```
**Вывод:**
```scss
Начинаем ждать...
(пауза 2 секунды)
Прошло 2 секунды — продолжаем!
```

#### time.After — таймаут в select (самое частое применение!)
Возвращает канал, который получит значение через указанное время. Идеально для установки таймаутов в `select`. Функция `time.After` — это стандартный способ задания таймаутов в конструкции `select`. Это позволяет ограничивать время выполнения операций: например, установить дедлайн для ответа от горутины, выполнения сложного вычисления или сетевого запроса. Следовательно, `time.After` предоставляет простой и эффективный механизм предотвращения бесконечного ожидания в конкурентных сценариях.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    fmt.Println("Ждём максимум 3 секунды...")
    
    select {
    case <-time.After(3 * time.Second):
        fmt.Println("Таймаут! Прошло 3 секунды")
    }
}
```
**Вывод:**
```scss
Ждём максимум 3 секунды...
(пауза 3 секунды)
Таймаут! Прошло 3 секунды
```

#### time.NewTimer — полноценный управляемый таймер
Возвращает структуру `*time.Timer`. Позволяет остановить таймер (`Stop()`) или сбросить (`Reset()`).
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    timer := time.NewTimer(5 * time.Second)
    defer timer.Stop()
    
    fmt.Println("Таймер запущен на 5 секунд...")
    <-timer.C
    fmt.Println("Таймер сработал!")
}
```
**Вывод:**
```scss
Таймер запущен на 5 секунд...
(пауза 5 секунд)
Таймер сработал!
```

#### time.AfterFunc — запуск функции через время
Запускает функцию в отдельной горутине через указанное время. Очень удобно для отложенных действий: отправка напоминаний, очистка кэша и т.д.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    fmt.Println("Запускаем отложенную задачу...")
    
    time.AfterFunc(2*time.Second, func() {
        fmt.Println("Привет из будущего! Прошло 2 секунды")
    })
    
    fmt.Println("Продолжаем основную работу...")
    time.Sleep(3 * time.Second) // чтобы увидеть результат
}
```
**Вывод:**
```erlang
Запускаем отложенную задачу...
Продолжаем основную работу...
(пауза 2 секунды)
Привет из будущего! Прошло 2 секунды
```

#### time.NewTicker — регулярные события (heartbeat, мониторинг, обновление)
Создаёт тикер, который отправляет текущее время в канал с заданным интервалом. Идеально для фоновых задач: проверка статуса, обновление данных, логирование каждые N секунд.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    ticker := time.NewTicker(1 * time.Second)
    defer ticker.Stop()
    
    fmt.Println("Тикер запущен — будет тикать каждую секунду")
    
    for i := 0; i < 5; i++ {
        t := <-ticker.C
        fmt.Printf("Тик! Сейчас: %s\n", t.Format("15:04:05"))
    }
    
    fmt.Println("Тикер остановлен")
}
```
**Вывод:**
```undefined
Тикер запущен — будет тикать каждую секунду
Тик! Сейчас: 14:22:10
Тик! Сейчас: 14:22:11
Тик! Сейчас: 14:22:12
Тик! Сейчас: 14:22:13
Тик! Сейчас: 14:22:14
Тикер остановлен
```

#### Округление и усечение времени — Round и Truncate
Иногда нужно «привести» время к началу часа, дня, месяца. `Truncate` отбрасывает «хвост», `Round` округляет до ближайшего значения.

Как указать точность округления:
1. `Truncate(d)` — отбрасывает всё, что меньше интервала `d`
    - Например, `Truncate(time.Hour)` отбрасывает минуты и секунды
    - `Truncate(24*time.Hour)` отбрасывает часы, минуты, секунды (начало дня)
2. `Round(d)` — округляет до ближайшего кратного интервала `d`
    - Если время посередине между двумя интервалами — округляется в большую сторону
    - Например, 12:31 при `Round(time.Hour)` станет 13:00

```css
package main

import (
    "fmt"
    "time"
)

func main() {
    now := time.Now()
    
    hourStart := now.Truncate(time.Hour)
    hourRounded := now.Round(time.Hour)
    
    fmt.Println("Текущее время:", now.Format("15:04:05"))
    fmt.Println("Начало текущего часа:", hourStart.Format("15:04:05"))
    fmt.Println("Округление до часа:", hourRounded.Format("15:04:05"))
}
```
**Вывод (пример):**
```undefined
Текущее время: 14:37:22
Начало текущего часа: 14:00:00
Округление до часа: 15:00:00
```

#### Unix-время: Unix() и UnixNano()
Классическое количество секунд (или наносекунд) с 1 января 1970 года — эпоха Unix. Очень часто используется в протоколах, токенах, базах данных.
```css
package main

import (
    "fmt"
    "time"
)

func main() {
    now := time.Now()
    fmt.Println("Unix-секунды:", now.Unix())
    fmt.Println("Unix-наносекунды:", now.UnixNano())
}
```
**Вывод (пример):**
```makefile
Unix-секунды: 1767820930
Unix-наносекунды: 1767820930123456789
```

#### Номер дня в году — метод YearDay()
Может быть удобно для расчёта статистики и отчётов.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    today := time.Now()
    fmt.Printf("Сегодня %d-й день года\n", today.YearDay())
    
    programmersDay := time.Date(today.Year(), 1, 256, 0, 0, 0, 0, time.UTC)
    fmt.Printf("День программиста в %d году: %s\n", today.Year(), programmersDay.Format("02.01.2006"))
}
```
**Вывод (пример):**
```1c
Сегодня 340-й день года
День программиста в 2025 году: 13.09.2025
```
