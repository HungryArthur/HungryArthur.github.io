В пакете `time` для работы с продолжительностью используется отдельный тип — `time.Duration`. Главное — не путать его с типом `time.Time`. Если `time.Time` обозначает конкретный момент (например, "12:30 1 января 2024"), то `time.Duration` описывает именно промежуток времени (например, "2 часа", "5 минут" или "10 секунд"). Длительность – просто **alias для int64**, где значение — количество наносекунд. Но Go делает его удобным для чтения и работы с помощью интуитивно понятных констант.

```go
type Duration int64
```

#### Как создать Duration?
Используйте и комбинируйте константы:
- `time.Nanosecond` — 1 нс
- `time.Microsecond` — 1 мкс
- `time.Millisecond` — 1 мс
- `time.Second` — 1 с
- `time.Minute` — 1 мин
- `time.Hour` — 1 час

```go
package main

import (
    "fmt"
    "time"
)

func main() {
    d := 2*time.Hour + 30*time.Minute + 15*time.Second + 500*time.Millisecond
    fmt.Println(d) // 2h30m15.5s
}
```
**Вывод:**
```undefined
2h30m15.5s
```

#### Как выводится и парсится time.Duration
Когда вы выводите `time.Duration` через `fmt.Println()` или `%v`, Go автоматически преобразует его в удобночитаемую строку, состоящую из комбинаций `h` (часы), `m` (минуты), `s` (секунды), `ms`, `µs`, `ns`. При этом:
- Go старается использовать **наиболее крупные единицы**, но не больше чем нужно.
- Если значение меньше секунды — показываются милли-, микро- или наносекунды.
- Если больше часа — появляются часы и минуты.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    fmt.Println(1500 * time.Millisecond)        // 1.5s
    fmt.Println(90 * time.Second)               // 1m30s
    fmt.Println(2*time.Hour + 30*time.Minute)   // 2h30m
    fmt.Println(3661 * time.Second)             // 1h1m1s
    fmt.Println(500 * time.Microsecond)         // 500µs
    fmt.Println(999 * time.Nanosecond)          // 999ns
}
```
**Вывод:**
```undefined
1.5s
1m30s
2h30m0s
1h1m1s
500µs
999ns
```

#### Парсинг Duration из строки
Функция `time.ParseDuration()` принимает ровно такой же формат, который выдаёт `fmt.Println`:
```go
package main

import (
	"fmt"
	"time"
)

func main() {
	d, err := time.ParseDuration("2h30m15s")
	if err != nil {
		fmt.Println("Ошибка:", err)
	}
	fmt.Println(d) // 2h30m15s

}
```
**Вывод:**
```undefined
2h30m15s
```

#### Операции с временем: прибавление и вычитание Duration
Для работы с временными интервалами у типа `time.Time` предусмотрены два ключевых метода: `Add` для прибавления длительности (`Duration`) к моменту времени и `Sub` для вычисления разницы между двумя моментами, которая возвращает значение типа `Duration`.

#### Прибавление: Add
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    now := time.Now()
    inTwoHours := now.Add(2 * time.Hour)
    fmt.Println("Сейчас:", now)
    fmt.Println("Через 2 часа:", inTwoHours)
}
```
**Вывод:**
```yaml
Сейчас: 2025-12-06 13:33:38.374463 +0300 MSK m=+0.000082792
Через 2 часа: 2025-12-06 15:33:38.374463 +0300 MSK m=+7200.000082792
```

#### Вычитание: Sub
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    start := time.Date(2025, time.December, 6, 0, 0, 0, 0, time.UTC)
    end := time.Date(2025, time.December, 6, 2, 30, 0, 0, time.UTC)
    duration := end.Sub(start)
    fmt.Println(duration) // 2h30m0s
}
```
**Вывод:**
```undefined
2h30m0s
```

#### Полезные функции: Since, Until
Для удобства работы с временными отрезками в пакете `time` также существуют две практичные функции:
- `time.Since(t)` — возвращает длительность (`Duration`), прошедшую с заданного момента `t` до текущего времени.
- `time.Until(t)` — возвращает длительность (`Duration`), оставшуюся от текущего времени до указанного момента `t`.

По сути, они являются удобными обёртками вокруг вызова `time.Now().Sub(t)` и `t.Sub(time.Now())` соответственно.
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    start := time.Now()
    time.Sleep(100 * time.Millisecond)
    elapsed := time.Since(start)
    fmt.Println("Прошло:", elapsed.Round(time.Millisecond))
    
    deadline := start.Add(1 * time.Second)
    remaining := time.Until(deadline)
    fmt.Println("Осталось:", remaining)
}
```
**Вывод:**
```makefile
Прошло: 100ms
Осталось: 900ms
```

#### Форматирование и парсинг времени — магия 2006-01-02 15:04:05
В Go для представлении даты используется **эталонная дата**:
```yaml
Mon Jan 2 15:04:05 MST 2006
```

Каждая часть этой даты — это шаблон:
- `2006` → год
- `01` или `Jan` → месяц
- `02` или `Mon` → день
- `15` → часы
- `04` → минуты
- `05` → секунды

#### Форматирование — метод Format
С помощью шаблона даты можно настроить её вывод в любом удобном виде. Есть готовый [список](https://pkg.go.dev/time#pkg-constants) константных форматов, которые можно использовать, если же в списке нет нужного формата, то с использованием эталонного времени можно реализовать свой формат вывода. Форматирование делается так:
```go
package main

import (
    "fmt"
    "time"
)

func main() {
    t := time.Now()
    
    fmt.Println("ISO дата:", t.Format("2006-01-02"))
    fmt.Println("Русский формат:", t.Format("02.01.2006"))
    fmt.Println("Полное время:", t.Format("2006-01-02 15:04:05"))
    fmt.Println("Красиво:", t.Format("02 Jan 2006, 15:04"))
    fmt.Println("RFC3339:", t.Format(time.RFC3339))
}
```
**Вывод (пример):**
```yaml
ISO дата: 2025-12-06
Русский формат: 06.12.2025
Полное время: 2025-12-06 13:47:06
Красиво: 06 Dec 2025, 13:47
RFC3339: 2025-12-06T13:47:06+03:00
```

#### Парсинг — функция Parse
То же самое работает и при чтении даты из строки. Чтобы программа правильно её распознала, нужно указать тот формат, в котором эта дата записана в строке. Для этого можно использовать готовые константы из пакета `time` или создать свой шаблон с помощью эталонной даты:
```go
package main

import (
	"fmt"
	"time"
)

func main() {
	layouts := []string{
		"2006-01-02",
		"02.01.2006",
		"2006-01-02 15:04:05",
		time.RFC3339,
		time.RFC3339,
	}

	dates := []string{
		"2025-12-06",
		"06.12.2025",
		"2025-12-06 14:22:10",
		"2025-12-06T14:22:10Z",
		"2025-12-06", // пример ошибочной даты для формата time.RFC3339,
	}

	for i, s := range dates {
		t, err := time.Parse(layouts[i], s)
		if err != nil {
			fmt.Println("Ошибка:", err)
			continue
		}
		fmt.Printf("Распознано: %v → %v\n", s, t.Format("02.01.2006 15:04"))
	}
}
```
**Вывод:**
```applescript
Распознано: 2025-12-06 → 06.12.2025 00:00
Распознано: 06.12.2025 → 06.12.2025 00:00
Распознано: 2025-12-06 14:22:10 → 06.12.2025 14:22
Распознано: 2025-12-06T14:22:10Z → 06.12.2025 14:22
Ошибка: parsing time "2025-12-06" as "2006-01-02T15:04:05Z07:00": cannot parse "" as "T"
```

#### Примеры наиболее распространенных готовых константы форматов:
- `time.RFC3339` → "2006-01-02T15:04:05Z07:00"
- `time.DateTime` → "2006-01-02 15:04:05"
- `time.DateOnly` → "2006-01-02"
- `time.TimeOnly` → "15:04:05"
