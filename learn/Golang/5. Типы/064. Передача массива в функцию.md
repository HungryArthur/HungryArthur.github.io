В языке программирования Go массивы, также как и остальные типы, могут передаваться в функции двумя способами: **по копии** и **по указателю**. При передаче по копии массив полностью копируется, и функция работает с его независимой копией. Изменения в функции не затрагивают исходный массив, что защищает данные, но требует дополнительной памяти. Это полезно, когда нужно сохранить оригинальные значения.

**Пример 1: Передача массива по копии**
Этот пример демонстрирует передачу массива в функцию по копии, где изменения внутри функции не влияют на оригинальный массив.
```go
package main

import "fmt"

func modifyArray(arr [3]int) {
    arr[0] = 999 // Изменяем первый элемент копии массива
    fmt.Println("Inside modifyArray:", arr)
}

func main() {
    array := [3]int{1, 2, 3}
    fmt.Println("Before modifyArray:", array)
    
    modifyArray(array)
    fmt.Println("After modifyArray:", array)
}
```
**Вывод:**
```less
Before modifyArray: [1 2 3]
Inside modifyArray: [999 2 3]
After modifyArray: [1 2 3]
```

**Пример 2: Передача массива по указателю**
Этот пример показывает, как передача указателя на массив или срез позволяет изменять оригинальные данные.
```go
package main

import "fmt"

func modifyArrayByPointer(arr *[3]int) {
    arr[0] = 999 // Изменяем первый элемент оригинального массива
    fmt.Println("Inside modifyArrayByPointer:", *arr)
}

func main() {
    // Передача массива по указателю
    array := [3]int{1, 2, 3}
    fmt.Println("Before modifyArrayByPointer:", array)
    modifyArrayByPointer(&array)
    fmt.Println("After modifyArrayByPointer:", array)
}
```
**Вывод:**
```less
Before modifyArrayByPointer: [1 2 3]
Inside modifyArrayByPointer: [999 2 3]
After modifyArrayByPointer: [999 2 3]
```

**Пояснение:**
В функции `modifyArrayByPointer` массив передаётся **по указателю** (`*[3]int`). Изменения через указатель затрагивают оригинальный массив.
