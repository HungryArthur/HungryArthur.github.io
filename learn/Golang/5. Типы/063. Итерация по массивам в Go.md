**Итерация по массиву** — это процесс последовательной обработки всех элементов массива, который позволяет выполнять вычисления, такие как подсчёт суммы, поиск максимума или вывод данных. В Go итерация по массиву чаще всего выполняется с использованием цикла `for`, который предоставляет гибкость в доступе к элементам по их индексам. Это особенно полезно для задач, где нужно обработать данные, например, подсчитать общее количество мест на трёх мероприятиях или заполнить массив данными, введёнными пользователем. Итерация требует осторожности, чтобы индексы оставались в пределах длины массива, иначе программа завершится с паникой во время выполнения.

**Простая итерация с использованием индексов** 
Использует цикл `for` с переменной индекса, которая изменяется от 0 до длины массива минус один. Этот подход даёт полный контроль над процессом, позволяя обращаться к элементам массива по их индексам. Например, для подсчёта общего количества мест на мероприятиях можно пройтись по массиву, суммируя значения всех элементов. Важно правильно задать границы цикла, чтобы избежать обращения по неверным индексам, что может привести к панике.

**Пример 1:**
```go
package main

import "fmt"

func main() {
	eventSeats := [3]int{100, 120, 80}
	totalSeats := 0
	for i := 0; i < 3; i++ {
		totalSeats += eventSeats[i]
	}
	fmt.Println("Total event seats:", totalSeats)
}
```
**Вывод:**
```csharp
Total event seats: 300
```

**Итерация с использованием range** 
Предоставляет более удобный способ перебора массива в Go. Ключевое слово `range` в цикле `for` возвращает индекс и значение каждого элемента, что упрощает доступ к данным и снижает вероятность ошибок с индексами. Этот подход полезен, когда нужно обработать элементы массива, например, вывести количество мест на каждом мероприятии с указанием их порядкового номера. Если индекс не нужен, его можно игнорировать, используя пустой идентификатор `_`.

**Пример 2:**
```go
package main

import "fmt"

func main() {
	venueCapacities := [3]int{90, 110, 95}
	for index, seats := range venueCapacities {
		fmt.Printf("Event %d has %d seats\n", index+1, seats)
	}
}
```
**Вывод:**
```vbnet
Event 1 has 90 seats
Event 2 has 110 seats
Event 3 has 95 seats
```

**Считывание элементов в массив в цикле** 
Разберем, как можно заполнить массив данными, введёнными пользователем, или другими значениями, поступающими во время выполнения программы. Это полезно, когда начальные данные неизвестны, например, когда количество мест на мероприятиях вводится организатором. Используя цикл `for`, можно последовательно считывать значения с помощью `fmt.Scan` и присваивать их элементам массива по индексам. Важно проверять корректность ввода и границы цикла, чтобы избежать ошибок или паники.

**Пример 4:**
```go
package main

import "fmt"

func main() {
	var seatCounts [3]int
	for i := 0; i < 3; i++ {
		fmt.Printf("Enter seats for event %d: ", i)
		fmt.Scan(&seatCounts[i])
	}
	fmt.Println("Entered seat counts:", seatCounts)
}
```
**Пример ввода:**
```undefined
95
115
80
```
**Вывод:**
```csharp
Entered seat counts: [95 115 80]
```
