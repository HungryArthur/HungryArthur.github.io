В этом шаге мы разберём, как безопасно выполнять преобразование типов в Go, используя проверку границ с помощью констант из пакета `math` и конструкции `switch` для обработки разных типов. Это позволяет избежать переполнения и сделать код надёжным.

**Константы пакета `math`:**
Пакет `math` предоставляет константы для минимальных и максимальных значений целочисленных типов, которые используются для проверки, находится ли значение в допустимом диапазоне перед преобразованием. Ниже приведена таблица с основными константами:

|Тип|Константа|Значение|Описание|
|---|---|---|---|
|`int8`|`math.MinInt8`|-128|Минимальное значение для знакового 8-битного целого|
|`int8`|`math.MaxInt8`|127|Максимальное значение для знакового 8-битного целого|
|`uint8`|`math.MaxUint8`|255|Максимальное значение для беззнакового 8-битного целого|
|`int16`|`math.MinInt16`|-32768|Минимальное значение для знакового 16-битного целого|
|`int16`|`math.MaxInt16`|32767|Максимальное значение для знакового 16-битного целого|
|`uint16`|`math.MaxUint16`|65535|Максимальное значение для беззнакового 16-битного целого|
|`int32`|`math.MinInt32`|-2147483648|Минимальное значение для знакового 32-битного целого|
|`int32`|`math.MaxInt32`|2147483647|Максимальное значение для знакового 32-битного целого|
|`uint32`|`math.MaxUint32`|4294967295|Максимальное значение для беззнакового 32-битного целого|
|`int64`|`math.MinInt64`|-9223372036854775808|Минимальное значение для знакового 64-битного целого|

**Безопасное преобразование с проверкой:**
Чтобы избежать переполнения, перед преобразованием типа нужно проверить, находится ли значение в допустимом диапазоне для целевого типа. Конструкция `switch` позволяет удобно обрабатывать разные типы преобразований, возвращая либо преобразованное значение, либо ошибку, если значение выходит за границы.

**Пример 1: Безопасное преобразование с использованием `switch`**
Этот пример показывает функцию `safeConvert`, которая проверяет границы перед преобразованием.
```go
package main

import (
	"fmt"
	"math"
)

func safeConvert(src int64, dstType string) (interface{}, error) {
	switch dstType {
	case "int8":
		if src < math.MinInt8 || src > math.MaxInt8 {
			return nil, fmt.Errorf("value %d is out of range for int8", src)
		}
		return int8(src), nil
	case "uint8":
		if src < 0 || src > math.MaxUint8 {
			return nil, fmt.Errorf("value %d is out of range for uint8", src)
		}
		return uint8(src), nil
	case "int16":
		if src < math.MinInt16 || src > math.MaxInt16 {
			return nil, fmt.Errorf("value %d is out of range for int16", src)
		}
		return int16(src), nil
	default:
		return nil, fmt.Errorf("unsupported type %s", dstType)
	}
}

func main() {
	defer fmt.Println("Conversion completed")
	
	value := int64(128)
	result, err := safeConvert(value, "int8")
	if err != nil {
		fmt.Println(err)
	} else {
		fmt.Printf("Converted to int8: %d\n", result)
	}
	
	value = int64(100)
	result, err = safeConvert(value, "uint8")
	if err != nil {
		fmt.Println(err)
	} else {
		fmt.Printf("Converted to uint8: %d\n", result)
	}
	
	value = int64(32768)
	result, err = safeConvert(value, "int16")
	if err != nil {
		fmt.Println(err)
	} else {
		fmt.Printf("Converted to int16: %d\n", result)
	}
}
```
**Вывод:**
```go
value 128 is out of range for int8
Converted to uint8: 100
value 32768 is out of range for int16
Conversion completed
```

**Пояснение:**
Функция `safeConvert` принимает значение типа `int64` и целевой тип (`"int8"`, `"uint8"`, `"int16"`). Используя `switch`, она проверяет, находится ли значение в допустимом диапазоне с помощью констант `math.MinInt8`, `math.MaxInt8`, `math.MaxUint8` и т.д. Если значение выходит за границы, возвращается ошибка. `defer` выводит `"Conversion completed"` в конце.

**Пример 2: Проверка преобразования с пользовательским вводом**
Этот пример демонстрирует безопасное преобразование на основе пользовательского ввода.
```go
package main

import (
	"fmt"
	"math"
)

func safeConvert(src int64, dstType string) (interface{}, error) {
	switch dstType {
	case "int8":
		if src < math.MinInt8 || src > math.MaxInt8 {
			return nil, fmt.Errorf("value %d is out of range for int8", src)
		}
		return int8(src), nil
	case "uint8":
		if src < 0 || src > math.MaxUint8 {
			return nil, fmt.Errorf("value %d is out of range for uint8", src)
		}
		return uint8(src), nil
	case "int16":
		if src < math.MinInt16 || src > math.MaxInt16 {
			return nil, fmt.Errorf("value %d is out of range for int16", src)
		}
		return int16(src), nil
	case "uint16":
		if src < 0 || src > math.MaxUint16 {
			return nil, fmt.Errorf("value %d is out of range for uint16", src)
		}
		return uint16(src), nil
	default:
		return nil, fmt.Errorf("unsupported type %s", dstType)
	}
}

func main() {
	var value int64
	var dstType string
	fmt.Scanln(&value)
	fmt.Scanln(&dstType)
	
	defer fmt.Println("Conversion completed")
	
	result, err := safeConvert(value, dstType)
	if err != nil {
		fmt.Println(err)
	} else {
		fmt.Printf("Converted value: %v\n", result)
	}
}
```
**Ввод:**
```go
100
int8
```
**Вывод:**
```yaml
Converted value: 100
Conversion completed
```
**Ввод:**
```go
300
uint8
```
**Вывод:**
```csharp
value 300 is out of range for uint8
Conversion completed
```

**Пояснение:**
Программа считывает значение `value` и целевой тип `dstType`. Функция `safeConvert` использует `switch` для выбора типа и проверки границ с помощью констант пакета `math`. Если преобразование безопасно, возвращается результат; иначе — ошибка. `defer` выводит `"Conversion completed"`.

**Рекомендации для безопасного преобразования:**
- Всегда проверяйте диапазон значений перед преобразованием, используя константы `math`.
- Используйте `switch` для обработки разных типов, чтобы код был структурированным.
- Возвращайте ошибки вместо выполнения небезопасных преобразований, чтобы избежать переполнения.
- Для больших чисел рассматривайте использование типов с большим диапазоном (например, `int64` вместо `int32`).
