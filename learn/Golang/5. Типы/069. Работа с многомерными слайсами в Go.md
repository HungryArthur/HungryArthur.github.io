Многомерные слайсы в Go — это слайсы, элементы которых сами являются слайсами, что позволяет создавать структуры, подобные многомерным массивам (например, матрицам). В отличие от массивов, слайсы динамичны, и их размер можно изменять с помощью `append`. Однако многомерные слайсы требуют явной инициализации, так как Go не выделяет память для них автоматически. Двумерный слайс, как и двумерный массив, часто называется **матрицей**.

#### Создание и инициализация
Для создания многомерного слайса, например двумерного, нужно сначала создать внешний слайс, а затем инициализировать каждый вложенный слайс. Используется `make` для выделения памяти и задания начальной длины/ёмкости.

Без инициализации вложенных слайсов доступ к элементам вызовет **панику**. Если мы обратимся по несуществующим индексам, то это также вызовет **панику**.

**Пример:**
```go
package main

import "fmt"

func main() {
	// Создаём двумерный слайс 3x4
	rows := 3
	cols := 4
	matrix := make([][]int, rows)
	for i := 0; i < rows; i++ {
		matrix[i] = make([]int, cols) // Инициализируем каждую строку
	}
	// Заполняем значения
	matrix[0][0] = 1
	matrix[1][2] = 5
	fmt.Println(matrix)
}
```
**Вывод:**
```lua
[[1 0 0 0] [0 0 5 0] [0 0 0 0]]
```

**Пояснение:** Внешний слайс создаётся с помощью `make([][]int, rows)`, затем каждая строка инициализируется с помощью `make([]int, cols)`. 

#### Итерация и доступ
Итерация и доступ к элементам многомерных слайсов выполняются аналогично массивам, используя вложенные циклы или индексы. Например, для матрицы `matrix[i][j]` обращение к элементу работает так же, как для массива `arr[i][j]`. Однако, в отличие от массивов, слайсы позволяют динамически изменять размер с помощью `append`.

**Пример итерации:**
```go
package main

import "fmt"

func main() {
	matrix := [][]int{{1, 2}, {3, 4}, {5, 6}}
	for i, row := range matrix {
		for j, val := range row {
			fmt.Printf("matrix[%d][%d] = %d\n", i, j, val)
		}
	}
}
```
**Вывод:**
```undefined
matrix[0][0] = 1
matrix[0][1] = 2
matrix[1][0] = 3
matrix[1][1] = 4
matrix[2][0] = 5
matrix[2][1] = 6
```

**Пояснение:** Вложенные циклы с `range` позволяют пройти по всем элементам. Слайсы могут иметь разную длину строк, что требует проверки длины вложенных слайсов при итерации.

#### Динамическое изменение
Многомерные слайсы можно расширять с помощью `append`, добавляя новые строки или элементы в существующие строки. Пакет `slices` (Go 1.21+) предоставляет функции, такие как `slices.Insert` или `slices.Delete`, для манипуляций с вложенными слайсами, но также это можно делать с помощью простых индексов. 

**Пример добавления строки:**
```go
package main

import (
	"fmt"
)

func main() {
	matrix := [][]int{{1, 2}, {3, 4}}
	newRow := []int{5, 6}
	matrix = append(matrix, newRow)
	fmt.Println(matrix)
}
```
**Вывод:**
```lua
[[1 2] [3 4] [5 6]]
```

**Пояснение:** `append` добавляет новый слайс как строку в матрицу.
