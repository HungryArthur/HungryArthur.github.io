Как вы уже знаете из [урока](https://stepik.org/lesson/1964832/step/13?unit=1992453), в языке Go синонимы типов (или пользовательские типы) создаются с помощью ключевого слова `type`, за которым следует имя нового типа и другой тип, для которого мы создаем синоним (например, `int`, `string`, `float64`). Такие типы, являются **отдельными типами**, несмотря на то, что они основаны на других типах, поэтому для них также можно реализовать методы.

- **Особенности:** Новый тип наследует свойства базового (например, арифметические операции для `int`), но считается независимым. Для синонима типа можно определять методы, как для структур, используя ресивер. 
- **Методы с указательным ресивером** также, как и для методов для структур, позволяют изменять значение переменной типа напрямую, так как работают с указателем на оригинальное значение, а не с его копией.
- **Применение:** Используются для валидации данных, добавления функциональности и улучшения читаемости кода.

Например, определение типа `BirthYear` на основе `int` позволяет создать метод для проверки, что год рождения > 1900, делая код более выразительным.

**Пример синонима типа с методом:**

```go
package main

import (
	"errors"
	"fmt"
)

type BirthYear int

func (y BirthYear) Validate() error {
	if y <= 1900 {
		return errors.New("invalid year")
	}
	return nil
}

func main() {
	var year BirthYear = 2000
	err := year.Validate()
	if err != nil {
		fmt.Printf("Ошибка: %v\n", err)
		return
	}
	fmt.Printf("Год валиден: %d\n", year)
}
```

**Вывод:**

```yaml
Год валиден: 2000
```

**Пояснение:** Тип `BirthYear` — синоним `int`. Метод `Validate` проверяет, что год > 1900, возвращая ошибку при несоответствии. В `main` создаётся экземпляр `BirthYear`, вызывается метод, и выводится результат.

**Пример с дополнительным методом:**
```go
package main

import (
	"errors"
	"fmt"
)

type BirthYear int

func (y BirthYear) Validate() error {
	if y <= 1900 {
		return errors.New("invalid year")
	}
	return nil
}

func (y BirthYear) Age(currentYear int) int {
	return currentYear - int(y)
}

func main() {
	var year BirthYear = 1995
	err := year.Validate()
	if err != nil {
		fmt.Printf("Ошибка: %v\n", err)
		return
	}
	age := year.Age(2025)
	fmt.Printf("Год рождения: %d, Возраст: %d\n", year, age)
}
```

**Вывод:**

```yaml
Год рождения: 1995, Возраст: 30
```

**Пояснение:** Метод `Age` вычисляет возраст, вычитая год рождения из текущего года. Это демонстрирует, как синоним типа позволяет добавлять функциональность, связанную с данными, делая код более семантически ясным.
#### **Пример с методом с указательным ресивером**
```go
package main

import (
	"errors"
	"fmt"
)

type BirthYear int

func (y BirthYear) Validate() error {
	if y <= 1900 {
		return errors.New("invalid year")
	}
	return nil
}

func (y *BirthYear) Set(newYear int) error {
	if newYear <= 1900 {
		return errors.New("invalid year")
	}
	*y = BirthYear(newYear) // Изменяем значение напрямую через указатель
	return nil
}

func main() {
	var year BirthYear = 1995
	err := year.Validate()
	if err != nil {
		fmt.Printf("Ошибка: %v\n", err)
		return
	}
	err = year.Set(2000)
	if err != nil {
		fmt.Printf("Ошибка: %v\n", err)
		return
	}
	fmt.Printf("Год рождения: %d\n", year)
}
```
**Вывод:**
```yaml
Год рождения: 2000
```

**Пояснение:** Тип `BirthYear` — синоним `int`. Метод `Validate` проверяет, что год > 1900. Метод `Set` с указательным ресивером изменяет значение года через `*y = BirthYear(newYear)`, если новый год валиден. В `main` создаётся `BirthYear`, проверяется, изменяется на 2000 и выводится результат.
