В предыдущих уроках мы изучили основы слайсов в Go, включая их создание с помощью `make`, итерацию с `range`, и динамическое изменение с помощью `append`. Слайсы — мощный инструмент для работы с последовательностями данных, и в Go существуют встроенные функции и пакеты, которые упрощают их обработку. В этой главе мы разберём некоторые функции из пакета `slices`, появившийся в Go 1.18

#### Пакет `slices`
Начиная с Go 1.18, стандартная библиотека включает пакет `slices`, который предоставляет удобные функции для работы со слайсами. Этот пакет упрощает такие операции, как поиск, сортировка, вставка и удаление элементов. Для использования пакета необходимо импортировать `"golang.org/x/exp/slices"` (в Go 1.21 и новее — `"slices"` в стандартной библиотеке). Ниже рассмотрены ключевые функции пакета `slices` с примерами.

**1. `slices.Contains(slice []T, value T) bool`**

Проверяет, содержит ли слайс указанный элемент. Работает с любым сравнимым типом `T`.

**Пример:**
```go
package main

import (
	"fmt"
	"slices"
)

func main() {
	s := []string{"apple", "banana", "cherry"}
	if slices.Contains(s, "banana") {
		fmt.Println("Слайс содержит banana")
	}
	if !slices.Contains(s, "orange") {
		fmt.Println("Слайс не содержит orange")
	}
}
```
**Вывод:**
```1c
Слайс содержит banana
Слайс не содержит orange
```

**Пояснение:** `slices.Contains` возвращает `true`, если элемент `"banana"` найден, и `false` для `"orange"`.

**2. `slices.Index(slice []T, value T) int`**

Возвращает индекс первого вхождения элемента в слайсе или -1, если элемент не найден.

**Пример:**

```go
package main

import (
	"fmt"
	"slices"
)

func main() {
	s := []int{10, 20, 30, 20}
	fmt.Printf("Индекс 20: %d\n", slices.Index(s, 20))
	fmt.Printf("Индекс 40: %d\n", slices.Index(s, 40))
}
```
**Вывод:**
```yaml
Индекс 20: 1
Индекс 40: -1
```

**Пояснение:** `slices.Index` возвращает 1 для первого вхождения `20` и -1 для отсутствующего `40`.

**3. `slices.Insert(slice []T, index int, values ...T) []T`**

Вставляет элементы `values` в слайс по указанному индексу, возвращая новый слайс. Оператор `...` позволяет передать несколько элементов.

**Пример:**
```go
package main

import (
	"fmt"
	"slices"
)

func main() {
	s := []int{1, 2, 3}
	s = slices.Insert(s, 1, 10, 20) // Вставляем 10 и 20 на позицию 1
	fmt.Printf("После вставки: %v\n", s)
}
```
**Вывод:**
```less
После вставки: [1 10 20 2 3]
```
**Пояснение:** `slices.Insert` добавляет элементы `10` и `20` на позицию 1, сдвигая остальные элементы вправо.


**ИЛИ**:
```go
package main

import (
	"fmt"
	"slices"
)

func main() {
	s := []int{1, 2, 3}
	s = slices.Insert(s, 1, 10, 20) // Вставляем 10 и 20 на позицию 1
	fmt.Printf("После вставки: %v\n", s)
}
```
**Вывод:**
```less
После вставки: [1 10 20 2 3]
```



**4. `slices.Delete(slice []T, i, j int) []T`**
Удаляет элементы с индекса `i` до `j-1` из слайса, возвращая новый слайс.

**Пример:**
```go
package main

import (
	"fmt"
	"slices"
)

func main() {
	s := []int{1, 2, 3, 4, 5}
	s = slices.Delete(s, 1, 3) // Удаляем элементы с индексов 1 и 2
	fmt.Printf("После удаления: %v\n", s)
}
```
**Вывод:**
```less
После удаления: [1 4 5]
```
**Пояснение:** `slices.Delete` удаляет элементы `2` и `3` (индексы 1 и 2), возвращая новый слайс.

**5. `slices.Sort(slice []T)`**

Сортирует слайс в порядке возрастания. Требует, чтобы тип `T` поддерживал сравнение (например, `int`, `float64`, `string`).

**Пример:**
```go
package main

import "fmt"

func main() {
    var a int
    var arr []int
    for range 5 {
        fmt.Scan(&a)
        arr = append(arr, a)
    }

    for i, j := 0, len(arr) - 1; i < j; i, j = i+1, j-1 {
        arr[i], arr[j] = arr[j], arr[i]
    }
    fmt.Println(arr)
}
```
**Ввод:**
```
[5 4 3 2 1]
```
**Вывод:**
```less
Отсортированный слайс: [1 2 3 4 5]
```

**Пояснение:** `slices.Sort` сортирует слайс в порядке возрастания, изменяя его на месте.

#### Практическое применение
Пакет `slices` и встроенные функции упрощают типичные задачи, такие как поиск, сортировка или изменение слайсов. Например, вместо ручной проверки наличия элемента можно использовать `slices.Contains`, а вместо сложных операций с `append` для вставки — `slices.Insert`. Эти инструменты делают код более читаемым и эффективным. Полный список функций пакета можно изучить в документации – [https://pkg.go.dev/slices](https://pkg.go.dev/slices).
