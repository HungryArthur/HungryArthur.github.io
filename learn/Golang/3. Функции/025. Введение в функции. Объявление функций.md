Функции позволяют структурировать код, делая его модульным, читаемым и повторно используемым. Вы узнаете, что такое функции, зачем они нужны, как их объявлять и как использовать параметры.

#### Что такое функции?

Функция в Go — это именованный блок кода, предназначенный для выполнения конкретной задачи. Она объединяет инструкции в единый модуль, который можно вызывать из разных частей программы. Каждая программа на Go начинается с функции `main`, которая является точкой входа. Функции объявляются с ключевым словом `func`, за которым следуют имя функции и тело в фигурных скобках `{}`. Функции могут выполнять действия, такие как вывод текста или вычисления, и делают код организованным, разбивая задачи на небольшие части.

Разберем объявление и использование простой функции на конкретном примере. В нем демонстрируется базовая структура функции в Go: функция `showMessage` объявляется с помощью ключевого слова `func`, не принимает параметров и не возвращает значений, а просто выводит сообщение при вызове. Функция `main` является точкой входа программы и вызывает `showMessage`, показывая, как одна функция может быть использована внутри другой для организации кода и избежания повторения одинаковых действий.

**Пример:**

```go
package main

import "fmt"

func showMessage() {
	fmt.Println("Это простая функция!")
}

func main() {
	showMessage()
}
```

**Вывод:**

```undefined
Это простая функция!
```
