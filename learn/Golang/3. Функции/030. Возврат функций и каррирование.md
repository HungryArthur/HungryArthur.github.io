Функциональный тип в Go — это способ описать, как выглядит функция: какие данные она принимает и что возвращает. Это как ярлык, который говорит: "Эта функция должна быть вот такого вида". Благодаря этому функции можно сохранять в переменные, передавать в другие функции или возвращать из них, как обычные числа или строки.

В Go вы можете указать функциональный тип прямо в переменной, например, `var myFunc func(int) int`. Это значит, что переменная `myFunc` может хранить функцию, которая принимает одно число и возвращает число. Можно также создать именованный тип с помощью `type`, например, `type MyFunc func(int) int`, но это не обязательно. Простой вариант с `var` часто удобнее, особенно если тип нужен только один раз. Если не проинициализировать функциональную переменную, то ее значение будет nil, а использование переменной будет вызывать **панику**. Подробнее о панике будет рассказано далее в курсе.

Функциональный тип помогает писать гибкий код. Например, вы можете сохранить функцию для удвоения числа, а потом заменить её на функцию для возведения в квадрат, если они обе принимают и возвращают одно число. Это позволяет менять поведение программы без изменения её основной части.

**Пример 1: Использование функционального типа без имени**

```go
package main

import "fmt"

func main() {
	var double func(int) int = func(x int) int {
		return x * 2
	}
	result := double(5)
	fmt.Printf("Результат: %d\n", result)
}
```
**Вывод:**
```makefile
Результат: 10
```

**Пояснение**: Переменная `double` объявлена с типом `func(int) int`, без именованного типа. Ей присваивается функция, которая удваивает число. Вызов `double(5)` показывает, что функция работает как обычное значение, которое можно сохранить в переменной.

**Пример 2: Именованный функциональный тип**
```go
package main

import "fmt"

type MathOp func(int, int) int

func add(a, b int) int {
	return a + b
}

func subtract(a, b int) int {
	return a - b
}

func main() {
	var op MathOp = add
	fmt.Printf("Сложение: %d\n", op(7, 4))
	op = subtract
	fmt.Printf("Вычитание: %d\n", op(7, 4))
}
```
**Вывод:**
```makefile
Сложение: 11
Вычитание: 3
```

**Пояснение**: Тип `MathOp` определяет функции, которые принимают два числа и возвращают одно. Переменная `op` сначала хранит `add`, потом `subtract`. Это показывает, как функциональный тип позволяет менять поведение программы, просто меняя функцию в переменной.
