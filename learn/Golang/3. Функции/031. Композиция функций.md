Композиция функций — это последовательное применение двух или большего числа функций, чтобы результат одной функции сразу шёл в другую. Например, если есть функции `f` и `g`, композиция `f(g(x))` означает, что сначала выполняется `g(x)`, а потом её результат передаётся в `f`.

В Go вы сами создаёте такую композицию, написав функцию, которая вызывает другие функции по очереди. Это удобно, когда нужно сделать несколько шагов обработки данных, например, сначала увеличить число, потом умножить его.

Композиция помогает писать код, который легко читать и использовать повторно. Вы можете комбинировать простые функции, чтобы получить сложные действия, не дублируя код.

**Пример 1: Простая композиция**
```go
package main

import "fmt"

func compose(f, g func(int) int) func(int) int {
	return func(x int) int {
		return f(g(x))
	}
}

func inc(x int) int {
	return x + 1
}

func double(x int) int {
	return x * 2
}

func main() {
	composed := compose(double, inc)
	result := composed(3)
	fmt.Printf("Результат: %d\n", result)
}
```
**Вывод:**
```go
Результат: 8
```

**Пояснение**: Функция `compose` создаёт новую функцию, которая сначала применяет `inc` (добавляет 1), а потом `double` (удваивает). Для `x=3`: `inc(3)=4`, `double(4)=8`. Переменные `inc` и `double` используют сигнатуру `func(int) int`.


**Пример 2: Композиция нескольких функций**
```go
package main

import "fmt"

func composeMultiple(fs ...func(int) int) func(int) int {
	return func(x int) int {
		result := x
		for _, f := range fs {
			result = f(result)
		}
		return result
	}
}

func double(x int) int {
	return x * 2
}

func triple(x int) int {
	return x * 3
}

func dec(x int) int {
	return x - 1
}

func main() {
	composed := composeMultiple(double, triple, dec)
	result := composed(2)
	fmt.Printf("Результат: %d\n", result)
}
```
**Вывод:**
```go
Результат: 11
```

**Пояснение**: Функция `composeMultiple` применяет функции по очереди: `double(2)=4`, `triple(4)=12`, `dec(12)=11`. Это показывает, как можно соединить несколько функций в цепочку, используя их как значения.
