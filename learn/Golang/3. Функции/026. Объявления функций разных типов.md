В предыдущем шаге мы рассмотрели базовую функцию без параметров и возвращаемых значений, которая выполняет простое действие. Теперь перейдём к функциям с параметрами, включая функции с несколькими параметрами одного типа, а также с именованными и неименованными возвращаемыми значениями.

Функции в Go объявляются с помощью ключевого слова `func`, за которым следуют имя функции (в сообществе Go принято использовать camelCase), параметры в круглых скобках `()`, возвращаемые типы (если есть) и тело в фигурных скобках `{}`. Правила Go требуют, чтобы открывающая фигурная скобка `{` находилась на той же строке, что и объявление функции.

#### Оператор return
Оператор `return` возвращает значения и прерывает выполнение функции. В функциях с неименованными возвращаемыми значениями `return` должен явно указывать возвращаемые значения. В функциях с именованными возвращаемыми значениями можно использовать `return` без аргументов — в этом случае возвращаются текущие значения этих переменных (возможно, нулевые, если возвращаемые значения не были инициализированы в функции).

Рассмотрим далее все возможные типы функций.

#### Функции с именованными параметрами и неименованным возвращаемым значением

Параметры указываются как `имя тип`, их значения передаются по значению, создавая копии внутри функции. Если функция возвращает одно неименованное значение, возвращаемый тип указывается без скобок: `func имя(имя_1 тип_1, имя_2 тип_2) тип {}`. Для возврата используется явный `return`, указывающий значение.

Рассмотрим функцию, которая принимает два числа и возвращает их сумму.

**Пример 1: Функция с именованными параметрами и неименованным возвращаемым значением**
```go
package main

import "fmt"

func addNumbers(a int, b int) int {
	return a + b
}

func main() {
	result := addNumbers(3, 4)
	message := fmt.Sprintf("Сумма: %d", result)
	fmt.Println(message)
}
```
**Вывод:**
```makefile
Сумма: 7
```

**Комментарий:** Функция `addNumbers` принимает два параметра типа `int` и возвращает их сумму через неименованное значение, используя явный `return`.

#### Функции с именованными параметрами и именованными возвращаемыми значениями

Именованные возвращаемые значения задаются в заголовке функции: `func имя(имя_1 тип_1, имя_2 тип_2) (имя_возвр_1 тип_1, имя_возвр_2 тип_2) {}`. Они автоматически инициализируются нулевыми значениями (например, `0` для `int`, `""` для `string`). Оператор `return` без аргументов возвращает эти значения, если они присвоены, или их нулевые значения, если не присвоены.

Рассмотрим функцию, которая выполняет деление и проверяет деление на ноль.

**Пример 2: Функция с именованными параметрами и именованными возвращаемыми значениями**
```go
package main

import "fmt"

func divideNumbers(a int, b int) (result int, err string) {
	if b == 0 {
		err = "Деление на ноль"
		return
	}
	result = a / b
	return
}

func main() {
	res, err := divideNumbers(10, 2)
	if err != "" {
		fmt.Println(err)
	} else {
		message := fmt.Sprintf("Результат деления: %d", res)
		fmt.Println(message)
	}
}
```
**Вывод:**
```undefined
Результат деления: 5
```

**Комментарий:** Функция `divideNumbers` использует именованные возвращаемые значения `result` и `err`. Простой `return` возвращает их значения, если они присвоены.
#### Функции с параметрами одного типа и неименованным возвращаемым значением

Если несколько параметров имеют один тип, его можно указать один раз: `a, b, c тип`. Для неименованных возвращаемых значений используется явный `return`: `func имя(имя_1, имя_2, ..., имя_n тип) тип {}`.

Рассмотрим функцию, которая суммирует три числа.

**Пример 3: Функция с параметрами одного типа и неименованным возвращаемым значением**

```go
package main

import "fmt"

func sumThree(a, b, c int) int {
	return a + b + c
}

func main() {
	result := sumThree(1, 2, 3)
	message := fmt.Sprintf("Сумма трёх чисел: %d", result)
	fmt.Println(message)
}
```
**Вывод:**
```undefined
Сумма трёх чисел: 6
```

**Комментарий:** Функция `sumThree` использует сокращённый синтаксис для параметров одного типа `a, b, c int` и возвращает неименованное значение через `return`.

#### Функции с параметрами одного типа и именованными возвращаемыми значениями

Комбинация сокращённого синтаксиса для параметров и именованных возвращаемых значений: `func имя(имя_1, имя_2, ..., имя_n тип) (имя_возвр_1 тип_1, имя_возвр_2 тип_2) {}`. `return` без аргументов возвращает именованные значения.

Рассмотрим функцию, которая вычисляет сумму и произведение двух чисел.

**Пример 4: Функция с параметрами одного типа и именованными возвращаемыми значениями**

```go
package main

import "fmt"

func calcTwo(a, b int) (sum int, product int) {
	sum = a + b
	product = a * b
	return
}

func main() {
	s, p := calcTwo(3, 4)
	message := fmt.Sprintf("Сумма: %d, Произведение: %d", s, p)
	fmt.Println(message)
}
```
**Вывод:**
```makefile
Сумма: 7, Произведение: 12
```

**Комментарий:** Функция `calcTwo` использует сокращённый синтаксис для параметров `a, b int` и именованные возвращаемые значения `sum` и `product`.

#### Функции с именованными параметрами и неименованными возвращаемыми значениями (множественные возвращаемые значения)

Если функция возвращает несколько неименованных значений, их типы указываются в скобках: `func имя(имя_1 тип_1, имя_2 тип_2) (тип_1, тип_2) {}`. `return` должен явно указывать все возвращаемые значения.

Рассмотрим функцию, которая проверяет, является ли число чётным.

**Пример 5: Функция с именованными параметрами и неименованными возвращаемыми значениями**

```go
package main

import "fmt"

func checkEven(n int) (int, string) {
	if n%2 == 0 {
		return n, "Чётное"
	}
	return n, "Нечётное"
}

func main() {
	num, status := checkEven(6)
	message := fmt.Sprintf("Число: %d, Статус: %s", num, status)
	fmt.Println(message)
}
```
**Вывод:**
```makefile
Число: 6, Статус: Чётное
```

**Комментарий:** Функция `checkEven` возвращает два неименованных значения, используя явный `return`.

#### Функции с именованными параметрами и именованными возвращаемыми значениями (множественные возвращаемые значения)

Комбинация именованных параметров и именованных возвращаемых значений: `func имя(имя_1 тип_1, имя_2 тип_2) (имя_возвр_1 тип_1, имя_возвр_2 тип_2) {}`. `return` может быть без аргументов, если значения присвоены.

Рассмотрим функцию, которая проверяет знак числа.

**Пример 6: Функция с именованными параметрами и именованными возвращаемыми значениями**

```go
package main

import "fmt"

func checkSign(n int) (value int, sign string) {
	value = n
	if n > 0 {
		sign = "Положительное"
	} else if n < 0 {
		sign = "Отрицательное"
	} else {
		sign = "Ноль"
	}
	return
}

func main() {
	val, s := checkSign(-4)
	message := fmt.Sprintf("Число: %d, Знак: %s", val, s)
	fmt.Println(message)
}
```
**Вывод:**
```makefile
Число: -4, Знак: Отрицательное
```

**Комментарий:** Функция `checkSign` использует именованные возвращаемые значения `value` и `sign`, возвращаемые через `return`.

#### Функции с неименованными параметрами

Крайне редкий случай использования непосредственно в функциях. Чаще всего это используется в функциональных типах, а также интерфейсах(подробнее об этих понятиях вы узнаете в следующих уроках). В этом случае функция принимает только конкретный тип, либо список типов, которые нельзя использовать в функции. В этом случае возможна любая комбинация возвращаемых значений, в функцию же при вызове необходимо передать корректный список аргументов правильных типов в нужной последовательности.

**Пример 7: Функция с неименованными параметрами, без возвращаемых значений**

```go
package main

import "fmt"

func showMessage(int, string, string) {
	fmt.Println("Это функция с неименованными параметрами!")
}

func main() {
	showMessage(5, "test", "test")
}
```
**Вывод:**
```undefined
Это функция с неименованными параметрами!
```

**Комментарий:** Функция `showMessage` использует неименованные параметры `int, string, string` и выводит строку на экран
