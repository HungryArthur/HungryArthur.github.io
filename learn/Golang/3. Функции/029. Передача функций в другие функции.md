Передача функций — это передача одной функции в качестве аргумента другой функции. Это позволяет создавать гибкие программы, где одна функция может работать с разными действиями, в зависимости от того, какую функцию ей передали.

В Go это удобно, потому что функции можно передавать как обычные значения. Например, вы можете написать функцию, которая применяет переданную ей функцию к числу или списку чисел. Это помогает не писать один и тот же код много раз, а просто менять функцию, которая делает основную работу.

Такой подход часто используют, чтобы обрабатывать данные разными способами. Например, вы можете передать функцию для удвоения чисел или для их возведения в квадрат, и основная функция будет работать с любой из них, если они подходят по формату.

**Пример 1: Передача функции для одного числа**

```go
package main

import "fmt"

func apply(op func(int) int, x int) int {
	return op(x)
}

func triple(x int) int {
	return x * 3
}

func main() {
	result := apply(triple, 4)
	fmt.Printf("Результат: %d\n", result)
}
```

**Вывод:**

```makefile
Результат: 12
```

**Пояснение:** Функция `apply` берёт функцию `op`, которая принимает число и возвращает число, и применяет её к числу `x`. Мы передаём функцию `triple`, которая утраивает число. Вызов `apply(triple, 4)` даёт 12, показывая, как можно передать функцию для выполнения действия.

**Пример 2: Передача функции для неопределенного числа переменных**

```go
package main

import "fmt"

func processList(op func(int) int, numbers ...int) int {
	result := 0
	for _, num := range numbers {
		result += op(num)
	}
	return result
}

func square(num int) int {
	return num * num
}

func main() {
	squared := processList(square, 1, 2, 3, 10)
	fmt.Printf("Результат: %v\n", squared)
}
```

Вывод:

```makefile
Результат: 114
```

**Пояснение:** Функция `processList` берёт список чисел и функцию `op`, применяя её к каждому числу. Мы передаём функцию `square`, которая возводит число в квадрат. Это показывает, как передача функций позволяет обрабатывать данные разными способами.
