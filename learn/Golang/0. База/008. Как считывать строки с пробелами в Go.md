В языке программирования Go часто нужно считывать строки, которые содержат несколько слов, разделённых пробелами, и обрабатывать их целиком до переноса строки (нажатие Enter). Это важно, например, при чтении пользовательского ввода или данных из файла. В этом уроке мы разберём, как правильно считывать такие строки.

#### Основы чтения строк

Для чтения строк в Go можно использовать функции из пакета `fmt`, такие как `Scan`, `Scanln` или `Scanf`. Они работают так же, как и с другими типами данных (например, числами): считывают отдельные слова, разделённые пробелами. Если вы вводите строку вроде `"Привет, мир!"`, эти функции разделят её на слова `"Привет,"` и `"мир!"`, что не всегда удобно, если нужна вся строка целиком.

Для чтения полной строки с пробелами до переноса строки лучше использовать библиотеку `bufio`. Она позволяет легко и надёжно считывать весь текст, включая пробелы.

#### Использование `bufio.Scanner`

Библиотека `bufio` предоставляет инструмент `bufio.Scanner`, который считывает данные построчно, сохраняя пробелы, до символа переноса строки (`\n`). Это идеальный способ для чтения строк, содержащих несколько слов, как единого текста.

**Что такое `os.Stdin`?**

`os.Stdin` — это поток ввода, который представляет данные, поступающие с клавиатуры (стандартный ввод). В Go стандартный ввод, вывод и ошибки доступны через пакет `os` как `os.Stdin` (ввод), `os.Stdout` (вывод) и `os.Stderr` (ошибки). Когда вы используете `bufio.NewScanner(os.Stdin)`, вы указываете, что хотите считывать данные, которые пользователь вводит с клавиатуры.

**Пример с `bufio.Scanner` для чтения с клавиатуры:**

```go

package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    fmt.Println("Введите строку с пробелами:")
    scanner := bufio.NewScanner(os.Stdin)
    scanner.Scan() // Считываем строку до переноса
    input := scanner.Text() // Получаем текст
    fmt.Println("Вы ввели:", input)
}
```

**Вывод (при вводе "Привет, мир!"):**

```undefined
Введите строку с пробелами:
Привет, мир!
Вы ввели: Привет, мир!
```

Ссылка на песочницу: [https://go.dev/play/p/0Y5l5YH6HhP](https://go.dev/play/p/0Y5l5YH6HhP)

**Как это работает:**

- `bufio.NewScanner(os.Stdin)` создаёт сканер для чтения из стандартного ввода (клавиатуры).
- `scanner.Scan()` считывает текст до переноса строки.
- `scanner.Text()` возвращает считанную строку, включая пробелы.

 **Пример с `bufio.Scanner` для чтения с клавиатуры нескольких строк:**

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    // Создаем ОДИН сканер для всего ввода
    scanner := bufio.NewScanner(os.Stdin)
    
    fmt.Println("Введите первую строку:")
    scanner.Scan() // Читаем первую строку
    line1 := scanner.Text()
    
    fmt.Println("Введите вторую строку:")
    scanner.Scan() // Читаем вторую строку (используем тот же сканер!)
    line2 := scanner.Text()
    
    fmt.Println("Введите третью строку:")
    scanner.Scan() // Читаем третью строку (опять тот же сканер!)
    line3 := scanner.Text()
    
    fmt.Printf("\nВы ввели:\n1: %s\n2: %s\n3: %s\n", line1, line2, line3)
}
```

**Исполнение программы:**

```makefile
Введите первую строку:
Привет
Введите вторую строку:
Мир
Введите третью строку:
Go

Вы ввели:
1: Привет
2: Мир
3: Go
```

Почему не нужно создавать новый сканнер каждый раз:

1. **Сканнер работает с потоком ввода** - он читает данные последовательно из `os.Stdin`
    
2. **Состояние сохраняется** - сканер запоминает текущую позицию в потоке
    
3. **Эффективность** - один сканер переиспользует буферы
    
4. **Корректная работа** - создание нового сканера может привести к:
    
    - Потере данных (если буфер был частично заполнен)
        
    - Непредсказуемому поведению
        

#### Почему использовать `bufio.Scanner`?

В отличие от функций `fmt.Scan*`, которые разбивают ввод на слова по пробелам, `bufio.Scanner` сохраняет строку целиком, как она была введена или записана в файле. Это делает его более удобным для задач, где нужно работать с полным текстом, например, с предложениями или командами.

#### Полезные советы

- Используйте `bufio.Scanner` для чтения строк с пробелами, будь то с клавиатуры (`os.Stdin`) или из файла.
- Всегда проверяйте ошибки при открытии файла (`err != nil`) и при сканировании (`scanner.Err()`).
- Если вы работаете с файлами, не забывайте закрывать их с помощью `file.Close()` или использовать `defer` для автоматического закрытия.
- Для других источников данных (например, сетевых соединений) `bufio.Scanner` также можно использовать, передав соответствующий поток ввода.
