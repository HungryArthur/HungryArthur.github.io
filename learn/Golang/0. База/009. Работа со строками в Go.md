Для лучшего **понимания** работы со строками в Go рассмотрим основные операции: соединение, сравнение, поиск подстрок и другие действия.

Строки в Go неизменяемы, поэтому операции создают новую строку, а не изменяют существующую. Это важно для оптимизации производительности при работе с большими данными.

```javascript
package main

import (
    "fmt"
    "strings"
)

func main() {
    // Простая конкатенация
    s1 := "Привет, "
    s2 := "Мир!"
    result := s1 + s2
    fmt.Println("С помощью +:", result)

    // Использование strings.Builder
    var builder strings.Builder
    builder.WriteString(s1)
    builder.WriteString(s2)
    fmt.Println("С помощью Builder:", builder.String())
}
```

**Вывод:**
```yaml
С помощью +: Привет, Мир!
С помощью Builder: Привет, Мир!
```

Для объединения(конкатенации) строк можно использовать оператор `+`, но для больших объёмов данных рекомендуется `strings.Builder` или `strings.Join` из пакета `strings` ([https://pkg.go.dev/strings](https://pkg.go.dev/strings)) для повышения эффективности.

#### Сравнение строк

Строки сравниваются с помощью операторов `==`, `!=`, `<`, `>`, `<=`, `>=`. Сравнение выполняется побайтно в UTF-8, что соответствует лексикографическому порядку.

```go
package main

import "fmt"

func main() {
    s1 := "Апельсин"
    s2 := "Яблоко"
    fmt.Println("s1 == s2:", s1 == s2)
    fmt.Println("s1 < s2:", s1 < s2)
}
```
**Вывод:**
```yaml
s1 == s2: false
s1 < s2: true
```

#### Поиск подстрок

Для поиска подстрок используются функции из пакета `strings`: `Contains` проверяет наличие подстроки, а `Index` возвращает индекс первого байта вхождения подстроки или `-1`, если подстрока не найдена.

```javascript
package main

import (
    "fmt"
    "strings"
)

func main() {
    text := "Привет, Мир!"
    fmt.Println("Содержит 'Мир':", strings.Contains(text, "Мир"))
    fmt.Println("Индекс 'Мир':", strings.Index(text, "Мир"))
}
```
**Вывод:**
```bash
Содержит 'Мир': true
Индекс 'Мир': 14
```
#### Другие полезные операции

Пакет `strings` предоставляет множество функций для работы со строками:

- `strings.ToUpper` и `strings.ToLower`: перевод строки в верхний или нижний регистр.
- `strings.TrimSpace`: удаление пробелов в начале и конце строки.
- `strings.Split`: разделение строки на слайс подстрок по разделителю.

```go
package main

import (
    "fmt"
    "strings"
)

func main() {
    text := "  привет, мир!  "
    fmt.Println("Верхний регистр:", strings.ToUpper(text))
    fmt.Println("Без пробелов:", strings.TrimSpace(text))
    fmt.Println("Разделение:", strings.Split("а,б,в", ","))
}
```

**Вывод:**
```less
Верхний регистр:   ПРИВЕТ, МИР!  
Без пробелов: привет, мир!
Разделение: [а б в]
```
#### Полезные советы

- Используйте `strings.Builder` для конкатенации большого количества строк, чтобы избежать лишних аллокаций памяти.
- При сравнении строк учитывайте, что UTF-8 чувствителен к регистрам и кодировке символов.
- Для работы с Unicode-символами преобразуйте строку в `[]rune`, если нужно обрабатывать отдельные символы.
