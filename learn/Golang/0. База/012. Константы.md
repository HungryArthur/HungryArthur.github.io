Константы в Go — это неизменяемые значения, объявляемые с помощью ключевого слова `const`. Они используются для хранения фиксированных данных, таких как математические константы, настройки программы или строковые значения, которые не должны изменяться. Константы вычисляются на этапе компиляции, что делает их эффективными и безопасными для использования в коде.

#### Объявление и использование констант

Константы объявляются с ключевым словом `const`, за которым следует имя и значение. Тип может быть указан явно или выведен автоматически. Константы нельзя изменить после объявления, и они всегда должны быть инициализированы.

**Пример 1: Простые константы**

Объявляем константы для числа Пи и приветствия.

```go
package main

import "fmt"

const pi = 3.14
const greeting = "Привет"

func main() {
    message := fmt.Sprintf("Сообщение: %s, Пи: %.2f", greeting, pi)
    fmt.Println(message)
}
```
**Вывод:**
```makefile
Сообщение: Привет, Пи: 3.14
```

**Комментарий:** Константы `pi` (float64) и `greeting` (string) используются в `fmt.Sprintf`. Типы выведены автоматически.

Ссылка на песочницу: [https://go.dev/play/p/v9OgIbFcYuq](https://go.dev/play/p/v9OgIbFcYuq)

**Пример 2: Константы с явным типом**

Объявляем константы с указанием типов.

```go
package main

import "fmt"

const maxRetries int = 3
const appVersion string = "1.0.0"

func main() {
    fmt.Println("Максимум попыток:", maxRetries)
    fmt.Println("Версия приложения:", appVersion)
}
```
**Вывод:**
```undefined
Максимум попыток: 3
Версия приложения: 1.0.0
```

**Комментарий:** Константы `maxRetries` (int) и `appVersion` (string) имеют явные типы, что полезно для строгой типизации.

**Групповое объявление констант**

Несколько констант можно объявить в блоке `const`, что удобно для связанных значений.

**Пример 3: Групповое объявление**

```go
package main

import "fmt"

const (
    minAge  = 18
    maxAge  = 100
    defaultAge = 30
)

func main() {
    fmt.Println("Минимальный возраст:", minAge)
    fmt.Println("Максимальный возраст:", maxAge)
    fmt.Println("Возраст по умолчанию:", defaultAge)
}
```
**Вывод:**
```1c
Минимальный возраст: 18
Максимальный возраст: 100
Возраст по умолчанию: 30
```

**Комментарий:** Блок `const` группирует связанные константы, улучшая читаемость кода.

**Интересный факт:** Нетипизированные константы в Go (например, `const x = 5`) могут использоваться с разными типами (int, float64), пока не привязаны к переменной. Это позволяет писать гибкий код, но требует осторожности при присваивании.

**Важно!**
- Попытка изменить константу (например, `pi = 3.15`) вызовет ошибку компиляции.
- Константы должны иметь значение при объявлении; `const x` недопустимо.
- Нетипизированные константы могут привести к неожиданным преобразованиям (например, `const x = 1.5` в `int` теряет дробную часть).

**Полезные советы:**
- Используйте константы для неизменяемых значений, таких как `const maxAttempts = 5`.
- Группируйте связанные константы в блок `const` для лучшей читаемости.
