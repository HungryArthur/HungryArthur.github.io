Пакет [strconv](https://pkg.go.dev/strconv) помогает делать эти преобразования безопасно и поддерживает работу с UTF-8, включая кириллицу и эмодзи в строках.

#### Основные функции пакета `strconv`
Вот функции, которые мы будем использовать:

| Функция               | Что делает                                     | Пример                                   | Результат |
| --------------------- | ---------------------------------------------- | ---------------------------------------- | --------- |
| `strconv.Atoi`        | Переводит строку в целое число (`int`).        | `strconv.Atoi("123")`                    | `123`     |
| `strconv.Itoa`        | Переводит целое число в строку.                | `strconv.Itoa(123)`                      | `"123"`   |
| `strconv.ParseFloat`  | Переводит строку в число с точкой (`float64`). | `strconv.ParseFloat("12.34", 64)`        | `12.34`   |
| `strconv.FormatFloat` | Переводит число с точкой в строку.             | `strconv.FormatFloat(12.34, 'f', 2, 64)` | `"12.34"` |

**Примечание**: Число `64` в `ParseFloat` и `FormatFloat` означает, что мы работаем с типом `float64`. В `FormatFloat` параметр `'f'` задаёт формат (обычный десятичный), а число `2` — количество цифр после точки.

#### Ошибки при преобразовании

Функции `Atoi` и `ParseFloat` возвращают два значения: результат и ошибку. Если строка не может быть преобразована в число (например, `"abc"` вместо `"123"`), функция возвращает ошибку. Мы пока не будем обрабатывать ошибки, так как это будет изучено позже. Вместо этого мы временно будем игнорировать ошибку, используя только корректные данные, используя `_`, но покажем, как выглядит ошибка, если ввести неправильные данные, а научимся обрабатывать ошибки в последующих уроках.

#### Примеры
**Пример 1: Перевод строки в число и обратно**

Считаем возраст из консоли, преобразуем его в число, прибавим 1 и преобразуем обратно в строку.

```go
package main

import (
    "bufio"
    "fmt"
    "os"
    "strconv"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)
    
    // Считываем возраст как строку
    scanner.Scan()
    age_str := scanner.Text()
    
    // Преобразуем строку в число
    age, _ := strconv.Atoi(age_str)
    
    // Прибавляем 1
    new_age := age + 1
    
    // Преобразуем число обратно в строку
    new_age_str := strconv.Itoa(new_age)
    
    // Форматируем сообщение
    message := fmt.Sprintf("Через год вам будет %s лет!", new_age_str)
    fmt.Println(message)
}
```
**Ввод:**
```undefined
20
```
**Вывод:**
```1c
Через год вам будет 21 лет!
```

**Пример 2: Работа с числом с точкой**

Считаем вес, преобразуем его в число, увеличим на 0.5 и преобразуем обратно в строку.

```go
package main

import (
    "bufio"
    "fmt"
    "os"
    "strconv"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)
    
    // Считываем вес как строку
    scanner.Scan()
    weight_str := scanner.Text()
    
    // Преобразуем строку в число с точкой
    weight, _ := strconv.ParseFloat(weight_str, 64)
    
    // Увеличиваем вес
    new_weight := weight + 0.5
    
    // Преобразуем число обратно в строку с 2 цифрами после точки
    new_weight_str := strconv.FormatFloat(new_weight, 'f', 2, 64)
    
    // Форматируем сообщение
    message := fmt.Sprintf("Новый вес: %s кг", new_weight_str)
    fmt.Println(message)
}
```

**Ввод:**
```undefined
70.2
```
**Вывод:**
```1c
Новый вес: 70.70 кг
```

**Пример 3: Что происходит при ошибке**
Если ввести неправильную строку вместо числа, программа может выдать ошибку или некорректный результат. Давайте посмотрим, что происходит, если вместо числа ввести буквы.

```go
package main

import (
    "bufio"
    "fmt"
    "os"
    "strconv"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)
    
    // Считываем возраст как строку
    scanner.Scan()
    age_str := scanner.Text()
    
    // Пытаемся преобразовать строку в число
    age, err := strconv.Atoi(age_str)
    
    // Выводим результат и ошибку
    message := fmt.Sprintf("Возраст: %d, ошибка: %v", age, err)
    fmt.Println(message)
}
```
**Ввод:**
```nginx
abc
```
**Вывод:**
```yaml
Возраст: 0, ошибка: strconv.Atoi: parsing "abc": invalid syntax
```

**Объяснение**: При вводе `"abc"` функция `Atoi` не может преобразовать строку в число, поэтому возвращает `0` и ошибку. Позже в курсе вы узнаете, как обрабатывать такие ошибки, чтобы программа работала надёжно.

#### Полезные советы:
- Считывайте строки в правильном порядке: имя, затем возраст, затем температура.
- Используйте `strconv.Atoi` для целых чисел, а `strconv.ParseFloat` для чисел с точкой.
- В `FormatFloat` указывайте нужное количество цифр после точки, чтобы вывод выглядел аккуратно.
- Если пользователь введёт буквы вместо цифр, программа может выдать ошибку. Позже вы узнаете, как это исправить.
- Проверяйте, что в `fmt.Sprintf` вы используете правильные спецификаторы для строк (`%s`).
- Пакет `strconv` работает только с ASCII-символами в числах, поэтому вводите цифры, а не буквы или эмодзи.
