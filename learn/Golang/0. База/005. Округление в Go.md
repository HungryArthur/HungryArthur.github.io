
Для выполнения различных видов **округления** в Go используйте функции из стандартного пакета `math`:
```go
package main

import ( "fmt" "math" )

func main() { num := 3.75 // Различные методы округления
	floor := math.Floor(num) // 3.0 (округление вниз)
	ceil := math.Ceil(num) // 4.0 (округление вверх)
	round := math.Round(num) // 4.0 (округление к ближайшему целому)
	trunc := math.Trunc(num) // 3.0 (отбрасывание дробной части)
	
	fmt.Println("Исходное число:", num)
	fmt.Println("Floor:", floor)
	fmt.Println("Ceil:", ceil)
	fmt.Println("Round:", round)
	fmt.Println("Trunc:", trunc) }
```

Для округления до определённого количества знаков после запятой используйте следующий подход:
```go
value := 3.14159265
rounded := math.Round(value*100) / 100 // Округление до 2 знаков
fmt.Println(rounded) // Вывод: 3.14
```
