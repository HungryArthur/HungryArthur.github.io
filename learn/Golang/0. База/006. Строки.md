
### Тип строка

В языке программирования Go строки представляют собой неизменяемые последовательности байтов, которые обычно интерпретируются как текст в кодировке UTF-8. Это означает, что каждая строка — это по сути массив чисел типа `byte` (синоним `uint8`).

Строки в Go особенные, потому что их нельзя изменить после создания. Если нужно что-то поменять, создается новая строка.

Строки в Go могут содержать буквы, цифры, символы на разных языках (например, кириллицу или иероглифы) и даже эмодзи, благодаря кодировке UTF-8. Это значит, что один символ может занимать от 1 до 4 байт в памяти компьютера. **Нулевое значение** для строки — это пустая строка (`""`).

#### Как записывать строки в Go?

В Go есть два способа записать строку:

1. **Обычные строки** — пишутся в двойных кавычках, например: `"Привет, мир!"`. В таких строках можно использовать специальные команды (escape-последовательности), чтобы добавить перенос строки или другие символы.
2. **Сырые строки** — пишутся в обратных кавычках, например: `` `Привет, мир!` ``. Они сохраняют текст точно как есть, без изменений, даже если там есть переносы строк или специальные символы.

**Пример:**

```go
package main

import "fmt"

func main() {
	// Пример обычной строки
	normalStr := "Это строка с \nпереносом строки"
	fmt.Println(normalStr)

	// Пример сырой строки
	rawStr := `Это сырая строка со множеством строк и без обработки \n как переноса`
	fmt.Println(rawStr)
}
```
**Вывод:**

```1c
Это строка с 
переносом строки
Это сырая строка со множеством строк и без обработки \n как переноса
```

#### Специальные команды (escape-последовательности)

В обычных строках (в двойных кавычках) можно использовать специальные команды, которые начинаются с символа `\`. Вот самые простые и полезные из них:

|Команда|Что делает|Пример|
|---|---|---|
|`\n`|Переносит текст на новую строку|`"Привет\nМир"` → Привет  <br>Мир|
|`\t`|Добавляет отступ (табуляцию)|`"Имя:\tТом"` → Имя: Том|
|`\"`|Добавляет кавычку в текст|`"Он сказал: \"Привет\""` → Он сказал: "Привет"|
|`\\`|Добавляет символ \|`"Путь: C:\\Go"` → Путь: C:\Go|
|`\u0420`|Добавляет символ Unicode (например, букву "Р")|`"\u0420"` → Р|
|`\U0001F600`|Добавляет эмодзи|`"\U0001F600"` → 😀|
**Пример с командами:**

```go
package main

import "fmt"

func main() {
    fmt.Println("Перенос:\nновая строка")
    fmt.Println("Табуляция:\t→")
    fmt.Println("Кавычки: \"Привет\"")
    fmt.Println("Путь: C:\\Go\\bin")
    fmt.Println("Эмодзи: \U0001F600")
}
```

**Вывод:**

```makefile
Перенос:
новая строка
Табуляция:	→
Кавычки: "Привет"
Путь: C:\Go\bin
Эмодзи: 😀
```

#### Почему строки в Go особенные?

Вот несколько важных вещей, которые нужно знать о строках в Go:

- **Нельзя изменить строку:** Если вы создали строку `"Привет"`, вы не можете поменять в ней буквы. Вместо этого создается новая строка.
- **Символы и байты:** В Go строка состоит из байтов. Например, английская буква занимает 1 байт, а русская буква или эмодзи — больше. Это важно, если вы хотите посчитать количество букв в строке.
- **Пустая строка:** Если строка не задана, она считается пустой (`""`). Это как пустой лист бумаги.


**Пример с подсчетом символов:**
```go
package main

import (
    "fmt"
    "unicode/utf8"
)

func main() {
    text := "日本語" // Японские символы
    fmt.Println("Сколько байт:", len(text))
    fmt.Println("Сколько символов:", utf8.RuneCountInString(text))
}
```

**Вывод:**
```undefined
Сколько байт: 9
Сколько символов: 3
```
В этом примере строка `"日本語"` содержит 3 символа, но занимает 9 байт, потому что японские символы "тяжелее", чем латиница и занимают по 3 байта.


**Индексация строк**

В Go индексация начинается с 0, то есть первый символ строки имеет индекс 0, второй — 1 и так далее. Однако, если вы хотите получить конкретный символ (например, букву или эмодзи), нужно использовать индекс от слайса рун (`[]rune(str)[0]`), а не просто `str[0]`, потому что `str[0]` вернет байт, а не символ. Как правильно работать с рунами и индексами, мы расскажем дальше. Поскольку строка - последовательность байтов(`uint8`), то для корректного представления числа в виде символа, который ему соответствует, полученный байт надо преобразовать в строку, либо руну.

```go
package main

import (
	"fmt"
)

func main() {
	s := "Привет 🚀"
	runes := []rune(s)

	fmt.Println("Символ с индексом 0:", string(runes[0]), ", без преобразования типа:", runes[0])
	fmt.Println("Символ с индексом 2:", string(runes[2]), ", без преобразования типа:", runes[2])
	fmt.Println("Символ с индексом 5:", string(runes[5]), ", без преобразования типа:", runes[5])
	fmt.Println("Символ с индексом 7:", string(runes[7]), ", без преобразования типа:", runes[7])
	fmt.Println("Байт с индексом 0:", string(s[0]), ", без преобразования типа:", s[0])
}
```

Вывод:

```yaml
Символ с индексом 0: П , без преобразования типа: 1055
Символ с индексом 2: и , без преобразования типа: 1080
Символ с индексом 5: т , без преобразования типа: 1090
Символ с индексом 7: 🚀 , без преобразования типа: 128640
Байт с индексом 0: Ð , без преобразования типа: 208
```

#### Полезные советы

- Если нужно проверить, пустая ли строка, просто сравните её с `""`, например: `str == ""`.
- Используйте сырые строки (в обратных кавычках), если хотите написать длинный текст с переносами строк или специальными символами без их обработки.
