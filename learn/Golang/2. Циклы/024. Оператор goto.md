Оператор `goto` в Go позволяет передать управление программы на метку (label) — именованную точку в коде. Метка обозначается именем, за которым следует двоеточие (`Label:`), а команда `goto Label` вызывает немедленный переход к этой метке. Это изменяет естественный поток выполнения программы, который обычно идёт сверху вниз.

`goto` может быть полезен в специфичных случаях, например, для выхода из вложенных циклов или обработки ошибок в низкоуровневом коде. Однако в современном программировании он редко используется и его стараются избегать, так как делает код менее читаемым и может привести к ["спагетти-коду"](https://ru.wikipedia.org/wiki/%D0%A1%D0%BF%D0%B0%D0%B3%D0%B5%D1%82%D1%82%D0%B8-%D0%BA%D0%BE%D0%B4).

В Go предпочтительны структурированные конструкции, такие как `if`, `for`, `break` и `continue`, чтобы не усложнить логику.

**Пример 1: Простое использование goto**

```go
package main

import "fmt"

func main() {
	var number int
	number = 5
	if number < 0 {
		goto Negative
	}
	fmt.Println("Число положительное или ноль")
	goto End
Negative:
	fmt.Println("Число отрицательное")
End:
	fmt.Println("Проверка завершена")
}
```

**Вывод:**
```1c
Число положительное или ноль
Проверка завершена
```

**Комментарий:** Если `number` – отрицательное, программа переходит к метке `Negative` с помощью `goto`, затем к `End`. Этот код можно переписать с помощью `if-else` для большей читаемости.

#### Использование goto с циклами

Оператор `goto` может использоваться для выхода из цикла или перехода к определённой точке программы, особенно если нужно обойти ограничения `break` (который прерывает только ближайший цикл).

**Пример 2: goto для выхода из цикла**

```go
package main

import "fmt"

func main() {
	var sum int
	var i int = 1
	for i <= 10 {
		sum += i
		if sum > 15 {
			goto Done
		}
		i++
	}
Done:
	message := fmt.Sprintf("Сумма: %d, последнее i: %d", sum, i)
	fmt.Println(message)
}
```
**Вывод:**
```less
Сумма: 21, последнее i: 6
```

**Комментарий:** Цикл суммирует числа от 1 до 10, но при `sum > 15` (после добавления 6) программа переходит к метке `Done`, прерывая цикл. Это аналогично `break`, но `goto` позволяет указать точку выхода.
