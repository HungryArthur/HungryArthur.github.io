В Go цикл `for` с конструкцией `range` используется для итерации по сложным структурам данных, таким как строки. Строка в Go — это последовательность байтов в кодировке UTF-8, и `for range` позволяет перебирать её символы (руны), возвращая их индекс и значение. Начиная с [Go 1.22](https://go.dev/blog/range-functions#forrange), `for range` также поддерживает итерацию по числовому диапазону (от 0 до n-1) и повторение действия n раз без использования счётчика. Этот подход универсален и по аналогии будет применяться к другим структурам данных, которые будут изложены позже в курсе, например, массивам, срезам и мапам.

**1. Итерация по строкам:**
Конструкция `for range` для строк имеет вид:

```go
for index, rune := range str {
    // index — позиция !байта! в строке
    // rune — символ (тип rune, Unicode-код)
}
```

Здесь `index` — начальная позиция символа, а `rune` — сам символ. Если индекс не нужен, его заменяют на `_`.

**Пример 1: Подсчёт гласных**
```go
package main

import "fmt"

func main() {
	text := "привет"
	count := 0

	fmt.Printf("Анализируем строку: \"%s\"\n", text)
	fmt.Println("=== ДЕБАГ ВЫВОД ===")

	for index, char := range text {
		// Дебаг вывод символа и индекса
		fmt.Printf("Индекс: %d, Символ: %c, Код: %U\n", index, char, char)

		// Проверяем русские гласные буквы
		if char == 'а' || char == 'е' || char == 'ё' || char == 'и' || char == 'о' ||
			char == 'у' || char == 'ы' || char == 'э' || char == 'ю' || char == 'я' {
			count++
			fmt.Printf("  -> Найдена гласная! Текущий счет: %d\n", count)
		}
	}

	fmt.Println("===================")
	message := fmt.Sprintf("Гласных букв в слове \"%s\": %d", text, count)
	fmt.Println(message)
}
```
**Вывод:**
```makefile
Анализируем строку: "привет"
=== ДЕБАГ ВЫВОД ===
Индекс: 0, Символ: п, Код: U+043F
Индекс: 2, Символ: р, Код: U+0440
Индекс: 4, Символ: и, Код: U+0438
  -> Найдена гласная! Текущий счет: 1
Индекс: 6, Символ: в, Код: U+0432
Индекс: 8, Символ: е, Код: U+0435
  -> Найдена гласная! Текущий счет: 2
Индекс: 10, Символ: т, Код: U+0442
===================
Гласных букв в слове "привет": 2
```

**Комментарий:** Цикл `for range` перебирает символы строки `text`. Для `"hello"` гласные — `e` и `o`, итого `count = 2`. 

**2. Числовой диапазон (Go 1.22+):**

Начиная с Go 1.22, `for range` поддерживает итерацию по числовому диапазону от 0 до n-1:
```go
package main

import "fmt"

func main() {
	const n = 5
	for i := range n {
		fmt.Print(i, " ")
	}
}
```
**Вывод:**
```undefined
0 1 2 3 4 
```

**Комментарий:** Цикл перебирает индексы от 0 до `n-1`. Это удобно для итераций, где нужен счётчик. 

**3. Повторение действия n раз (Go 1.22+):**

Если счётчик не нужен, можно использовать `for range n` для повторения действия n раз:
```go
package main

import "fmt"

func main() {
	const n = 5
	for range n {
		fmt.Print("+")
	}
}
```
**Вывод:**
```diff
+++++
```

**Комментарий:** Цикл выполняет действие 5 раз без использования счётчика. Это упрощает код для повторяющихся операций.
